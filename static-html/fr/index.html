Documentation Perl version [% perl_version %]

Contenu

fr.perldoc.org contient la documentation de Perl version [% perl_version %], aux formats HTML et PDF.

Pour voir les nouveautés de Perl [% perl_version %], lire la page de manuel perldelta.

Si vous débutez en Perl, vous devriez commencer par la lecture de la page de manuel perlintro qui est une introduction générale destinée aux débutants et qui fournit les éléments nécessaires à une bonne navigation dans la documentation complète de Perl. La FAQ vous fournira aussi des réponses à plus de 300 questions fréquemment posées.

Fonctionnalités du site

Recherche rapide des pages

Pour une navigation aisée, saisissez n'importe quelle fonction Perl, page de manuel ou nom de module dans le champ de recherche pour aller directement à la section appropriée (les résultats de la recherche seront aussi disponible).

Coloration syntaxiques et liens dans les exemples

La documentation Perl contient des milliers d'exemples de code Perl. Ces exemples sont colorés syntaxiquement, et les fonctions / noms de module ont des liens vers leur documentation respective. Par exemple : 

#! /usr/bin/perl

use strict;
use warnings;

my $greeting = "Hello, World!";
print $greeting,"\n";  # Say hello

Etiquettes

Chaque page ou recherche sur ce site peut être étiquettée en sélectionnant le lien "Ajouter cette page" dans la barre d'outils en haut à droite de la page.

Ajuter une étiquette crée un lien dynamique vers le document dans la section Etiquettes de la barre d'outils, permettant d'accéder aux pages fréquemment visitées en un seul clic depuis n'importe où dans le site.

Affichage personnalisable

La barre d'outils et le menu de navigation peuvent être cachés en cliquant sur l'icone fermer () correspondante en haut de la page.

Les préférences d'affichage et les détails des étiquettes sont stockés dans un cookie sur votre navigateur, ils seront donc préservés quand vous visiterez ce site depuis le même ordinateur.

Téléchargements

La documentation complète est téléchargeable pour un usage hors connexion.

Version complète - contient les fichiers HTML et PDF

http://perldoc.perl.org/perldoc.tar.gz (24Mb).

Version allégée - contient les fichiers HTML seulement

http://perldoc.perl.org/perldoc-html.tar.gz (3Mb).

De même que les pages de documentation, l'outil de recherche perldoc est aussi inclu dans les téléchargements ci-dessus. Aucune installation n'est requise, il suffit de décompresser l'archive et d'ouvirer le fichier index.html dans votre navigateur web.

Pour télécharger Perl, veuillez aller à http://www.perl.org/get.html.
